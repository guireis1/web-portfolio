Indirect methodfor Nadal

Regression with accelerometers

This project aims to implement a Machine Learning (ML) model capable of providing the Nadal value from information given by cheaper and easier-to-handle sensors such as inertial sensors. From the dynamic simulation of the rail car, the Nadal value and the accelerations of the car were obtained. From these two data sets, the ML model is trained and validated to recognize the Nadal value from unseen wagon accelerations.A comparison between different ML models, the acceleration measurement location and different inputs were tested to see which combination would provide the best correlation.

It was part of my graduation project. The actual accelerometer data was provided by the company VALE, and the simulated data was made at the UFES laboratory.
